# Excel Merger - Объединение Excel файлов

[![Go Version](https://img.shields.io/badge/Go-1.16%2B-blue.svg)](https://golang.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Программа для объединения нескольких Excel файлов (`.xlsx`, `.xls`) в один с сохранением форматирования, стилей, формул и гиперссылок.




##  Установка и запуск
1. Скачайте файл "Exsel-combine.exe" и можете запустить его двойным нажатие левой кнопки мыши.
2. После запуска появится окно выбора файлов. В нём выберите файлы которые необходимо объединить и нажмите кнопку открыть.
3. После чего окно закроется и когда закончит откроет окно и покажет объединёный файл.
